## 1. Plataformas Recomendadas

### Generales

- **[Class Central](https://www.classcentral.com/)**: Agregador de cursos de múltiples plataformas.
- **[CodeSignal](https://codesignal.com/learn/course-paths)**: Plataforma para aprendizaje y evaluación técnica.
- **[FreeCodeCamp](https://www.freecodecamp.org/learn)**: Cursos interactivos gratuitos para desarrolladores.
- **[MIT OpenCourseWare](https://ocw.mit.edu/)**: Material completo de cursos de MIT, sin registro y descargable.
- **[Roadmap.sh](https://roadmap.sh/)**: Rutas de aprendizaje visuales por rol o tecnología (Frontend, Backend, DevOps, AI, etc).
- **[Exercism](https://exercism.org/)**: Práctica de programación con mentoría humana en más de 80 lenguajes.

### Inteligencia Artificial y Data Science

- **[Kaggle](https://www.kaggle.com/learn)**: Cursos y competiciones para Data Science y Machine Learning.
- **[Cognitive Class](https://cognitiveclass.ai/)**: Cursos gratuitos en Data Science e IA proporcionados por IBM.
- **[Hugging Face](https://huggingface.co/learn)**: Recursos educativos sobre modelos de Machine Learning y PLN.
- **[Claude](https://claude.com/resources/courses)**: Recursos y guías proporcionadas por Anthropic.
- **[Fast.ai](https://course.fast.ai/)**: Curso práctico de Deep Learning, basado en su libro disponible gratis en línea.
- **[AMD AI Academy](https://developer.amd.com/)**: Recursos para el desarrollo de IA con tecnologías de AMD.
- **[Santander AI resources](https://github.com/SantanderAI)**: Recursos open source de Santander para aplicar IA a deteccion de Fraude y actividades fiancieras

### Redes, Infraestructura y Sistemas

- **[Networking Academy](https://www.netacad.com/)**: Cursos de redes y ciberseguridad impartidos por Cisco.
- **[Linux Foundation](https://training.linuxfoundation.org/free-courses/)**: Cursos gratuitos sobre Linux y tecnologías Open Source.

### Git y Control de Versiones

- **[Pro Git](https://git-scm.com/book)**: Libro completo gratuito, incluye videos cortos introductorios.
- **[GitHub Skills](https://skills.github.com/)**: Cursos interactivos integrados en GitHub con retroalimentación automática.
- **[W3Schools: Git Tutorial](https://www.w3schools.com/git/)**: Tutorial con ejemplos para GitHub, GitLab y Bitbucket, sin necesidad de cuenta.
- **[Microsoft Learn: Introduction to Git](https://learn.microsoft.com/en-us/training/modules/intro-to-git/)**: Módulo introductorio gratuito.

### Diseño UI/UX

- **[Figma Learn](https://help.figma.com/hc/en-us/categories/360002042354-Figma-Learn)**: Cursos oficiales de Figma, utilizando únicamente el plan gratuito Starter.
- **[Figma Learn Design](https://www.figma.com/community)**: Clases breves sobre fundamentos de diseño con ejercicios prácticos.
- **[Alison: Introduction to UI/UX Design](https://alison.com/course/introduction-to-ui-ux-design)**: Curso introductorio gratuito con certificado opcional.
- **[Google UX Design (Coursera)](https://www.coursera.org/professional-certificates/google-ux-design)**: Certificado profesional, con posibilidad de ayuda financiera.

### Mecanografía

- **[TypingClub](https://www.typingclub.com/)**: Cursos de mecanografía táctil con lecciones, juegos y videos, disponible en español.
- **[Keybr](https://www.keybr.com/)**: Práctica enfocada en velocidad y precisión, genera texto adaptado a tus errores de tecleo.
- **[Typing Study](https://www.typingstudy.com/)**: Curso completo de mecanografía táctil con tests de velocidad.
